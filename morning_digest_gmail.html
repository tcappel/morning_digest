⚙ One-time setup required This app needs a free Google API key to read your Gmail. It takes ~3 minutes:
  1. Go to console.cloud.google.com and create a project
  2. Enable the Gmail API for that project
  3. Go to APIs & Services → Credentials → Create Credentials → OAuth 2.0 Client ID
  4. Choose Web application, add http://localhost and the URL of this file to Authorized Origins
  5. Copy your Client ID and paste it below, then click Connect Gmail
